NormalVisuSR_all

报表展示可视化模块-静态图片

Tag:

  • R

Param:

  • x_data_factor_flag (string) : 是否转化成维度因子,输入"y"/"Y"执行。例:x_data_fsctor_flag<-"y"
  • reorder_flag (string) : 是否按y轴排序,"order"升序，"desc"降序,其他不排序。例:reorder_flag<-"order"
  • lab_title (string) : 图表标题内容。
  • x_lab_title (string) : x轴标题内容
  • y_lab_title (string) : y轴标题内容。

Input:

  • df (csv)

Output:

  • pp1 (pdf)
